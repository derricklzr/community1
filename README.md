# 仿知乎交友社区
社区主要功能：项目基于SSM框架，使用 Spring Boot 搭建，使用到了Redis，Kafka, Elasticsearch等框架。核心功能为：

未登录时：登录注册功能；帖子搜索功能；查看帖子功能；

已登录时：社区发帖、点赞、回复功能；私信交互功能；系统通知功能；个人主页及设置功能；帖子搜索功能；用户权限管理功能等
